# KiCad Project Area

Use this folder for the hardware design files as they are created.

Suggested contents once work starts:

- project schematic files
- symbol and footprint library notes if needed
- PDF schematic exports for review
- manufacturing outputs only if you want them versioned here

Suggested naming:

- project name: `bean-pounder-hw` or `elizabeth-central-command-hw`
- revision tags: `rev-a`, `rev-b`, etc.

Before starting layout, make sure the schematic decisions in the parent `hardware/` documents are still aligned with the firmware.
